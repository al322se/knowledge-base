# Knowledge Base Index

Layer 0 routing map. Keep this file short and use links to disclose detail progressively.

## How To Use

1. Start here.
2. Open the smallest relevant repo or system page.
3. Expand into architecture pages only when the question crosses repositories.
4. Open source files only when a claim needs verification.
5. Use raw transcripts as historical evidence, not as verified knowledge.
