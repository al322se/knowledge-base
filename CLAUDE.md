# Инструкции для Claude Code

Основной источник правил для агентов: [AGENTS.md](AGENTS.md).

Перед работой с базой знаний:

1. Открой [KB_INDEX.md](KB_INDEX.md).
2. Затем открой минимальную нужную страницу.
3. Следуй правилам проходок, provenance, `[требует уточнения]`, progressive disclosure и единого источника истины из `AGENTS.md`.
4. Не дублируй знания между страницами; добавляй ссылки с пояснением, что по ссылке и зачем читать.

Этот файл нужен как стандартная точка входа для Claude Code. Правила не дублируются здесь, чтобы `AGENTS.md` оставался единым источником истины.
