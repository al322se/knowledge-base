# Выжимки источников

Выжимка - короткий слой между сырым источником и основной базой знаний.

## Что извлекать

- новые факты;
- бизнесовые правила;
- технические решения;
- пользовательские предпочтения;
- утверждения, которые требуют проверки;
- страницы базы знаний, которые надо обновить.

Используй [выжимку источника](../../шаблоны/источник-выжимка.md).
